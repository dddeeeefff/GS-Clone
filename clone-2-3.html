모바일 App. 회원가입 및 사용방법

에너지플러스EV 사용방법을 알려드립니다.

HOME EV 충전정보 모바일 App. 회원가입 및 사용방법

1.로그인 클릭 후 회원가입 버튼 클릭

2.약관 전체동의 체크 후 하단 동의버튼 클릭

3.휴대전화 인증 클릭 후 본인인증 진행

4.인증 완료 후 본인인증 진행

5.정보 입력 후 확인을 누르면 회원가입 완료

6.로그인 후 하단 전체메뉴 선택

7.결제수단 클릭하기

1.전체메뉴 클릭 후 회원카드 메뉴 클릭

2.카드신청/등록 버튼 클릭

3.회원카드 발급신청 버튼 클릭

4.패스워드 입력 후 개인정보 입력

5.발급요청 클릭하면 신청완료(신청 후 4~5 영업일 내 우편발송되니 우편함 꼭 확인해주세요)

6.회원/제휴카드 등록 버튼 클릭

7.발급받은 실물카드 정보 입력 후 회원/제휴카드 등록 버튼 클릭 회원카드 등록 완료

8.홈 화면 상단 바로충전등록 버튼 클릭

9.회원카드 설정 후 다음단계 버튼 클릭

10.충전방식/충전량 설정 후 별칭(기억하기 쉬운 이름)입력 다음단계 버튼클릭

11.포인트 사용 설정 후 다음단계 버튼 클릭

12.결제카드 설정 후 바로저장 클릭

1.앱 메인 화면에서 NFC 아이콘 선택

2.충전카드관리 화면에서 회원카드신청/등록 선택

3.카드신청/등록 화면에서 온라인회원카드 발급하기(NFC) 선택

4.개인정보활용동의 안내 팝업에서 동의 선택

5.온라인카드 패스워드 입력

6.신청인 이름 입력

7.카드번호 확인

8.회원카드 발급완료 팝업창 확인

1.앱 실행 후 충전시작 버튼 클릭

2.충전기 화면에서 충전시작 버튼 클릭

3.회원카드 버튼 터치

4.휴대폰 바코드를 충전기기 바코드란에 스캔

5.[커넥트]충전기 연결

6.앱으로 충전완료

7.충전완료 후 영수증 확인
